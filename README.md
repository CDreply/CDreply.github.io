# 沉答Ai小游戏乐园 🎮

> 余风清韵与WorkBuddy合作打造的趣味小游戏合集

## 🌐 在线访问

**官方网站**：https://cdreply.github.io

## 📱 特色游戏

### 🧩 沉淀方块
- **游戏类型**：益智、策略
- **特色功能**：
  - 实时时间计时（每秒自动刷新）
  - 空格键使用统计（按键即更新）
  - 炸弹道具系统（概率随等级提升）
  - Web Audio API合成音效
  - 响应式设计（PC/平板/手机）
- **操作方式**：
  - 键盘：方向键移动/旋转，空格快速下落
  - 触摸：滑动操作（移动端）
  - 快捷键：P暂停，B炸弹

### 🐍 贪吃蛇
- **游戏类型**：休闲、闯关
- **特色功能**：
  - 多种道具：金苹果、冰冻、双倍积分等
  - 关卡升级系统
  - 移动端手势操作
  - 粒子特效系统
  - 个性化音效
- **操作方式**：
  - 键盘：方向键或WASD
  - 触摸：滑动操作（移动端）

## 🚀 快速开始

### 本地运行
```bash
# 进入项目目录
cd github-pages

# 启动本地服务器
python -m http.server 5500

# 打开浏览器访问
# http://localhost:5500/
```

### 部署到GitHub Pages
1. 将所有文件上传到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择main分支的根目录
4. 访问 https://cdreply.github.io

详细部署步骤请查看 [DEPLOYMENT_GUIDE.md](DEPLOYMENT_GUIDE.md)

## 🛠️ 技术栈

| 技术 | 用途 | 说明 |
|------|------|------|
| HTML5 Canvas | 游戏渲染 | 所有游戏画布渲染 |
| JavaScript | 游戏逻辑 | 完整的游戏实现 |
| Web Audio API | 音效系统 | 不依赖外部音频文件 |
| CSS3 | 样式和动画 | 响应式设计 |
| localStorage | 数据存储 | 保存游戏进度和高分 |
| Jekyll | 静态站点 | GitHub Pages支持 |

## 🎯 设计理念

### 用户体验
- **简洁直观**：界面清晰，操作简单
- **响应迅速**：实时反馈，无延迟感
- **跨平台**：PC、平板、手机全适配
- **无障碍**：键盘和触摸双重支持

### 技术特色
- **无依赖**：单文件HTML，无需外部库
- **高性能**：优化渲染，流畅运行
- **离线可用**：无需网络连接即可游戏
- **现代标准**：使用最新Web技术

### 游戏设计
- **中文优先**：界面全部使用中文
- **音效定制**：每种游戏有独特音效
- **进度保存**：自动保存游戏记录
- **难度平衡**：精心调整的游戏平衡
